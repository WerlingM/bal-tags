# Break-a-Leg Tags — Documentation

*Credit to Jason McCoy for the original design and years of supporting the SBHS Running Dog Productions*

## Process overview

Each show goes through four stages:

1. **[Materials and Preparation](materials.md)** — cutting plywood sheets to printer size
2. **[Art Prep](art-prep.md)** — test printing, layout, and generating print sheets
3. **[Print](print.md)** — UV printing front and back on the Roland LEF2-200
4. **[Cut](cut.md)** — laser cutting badges on the Thunder Nova 51
